## שלב 0 — אמת מידה: בנצ'מרק אמיתי (~8 שעות) 🎯 החשוב ביותר

בלי מספר אמיתי אין עסקה. כל השאר בא אחרי.

- [x] **0.1** (בוצע — `tests/real_benchmark.py`) תשתית בנצ'מרק: סקריפט שמקבל תיקיית סרטונים + קובץ תוויות (AI/אמיתי), מריץ את הצינור המלא, ומפיק דוח: precision, recall, FPR, מטריצת בלבול, פירוק לפי פלטפורמה ולפי שכבת-הכרעה
- [x] **0.2** (הפרוטוקול כתוב — `tests/COLLECTION_PLAN.md`; האיסוף עצמו אצל ניתאי) פרוטוקול איסוף חצי-אוטומטי: רשימת 300 סרטוני יעד (150 AI / 150 אמיתי) מטיקטוק/יוטיוב/אינסטגרם, כולל מקרים קשים: קמח/קונפטי/ספריי, מצלמות אבטחה, וידאו ישן, סלואו-מושן. ההורדה דרך הטלפון של ניתאי (IP ביתי) עם ה-Share Intent הקיים
- [ ] **0.3** ריצה + דוח + תיקון הכשלים שיתגלו (לולאה: מדוד → תקן → מדוד)
- [~] **0.4** דף `/accuracy` ציבורי עם מתודולוגיה ומדדי ה-CV הכנים ✅ — יתעדכן עם תוצאות הבנצ'מרק העיוור כשירוץ

**✔ מושלם כאשר:** יש דוח על ≥200 סרטונים אמיתיים עם precision ≥90% ו-FPR ≤5%, מפורסם באתר.

## שלב 1 — המודל: מדיוק טוב לדיוק מכירתי (~12 שעות)

- [x] **1.1a** הסרת נתיבי macOS קשיחים מ-`train_forever.py` — הדאטהסט מוגדר ב-`VERIFAI_DATASET_DIR` ✅
- [x] **1.1b-קוד** workflow אימון בענן (`.github/workflows/train.yml`): ריצה מתוזמנת/ידנית, תוצאות לענף `auto-train` + artifacts ✅ — נותר: להגדיר secret של PEXELS_API_KEY ולהריץ עד 2,000+ דגימות
- [ ] **1.2** איזון מקורות: לוודא שהמודל לומד "AI" ולא "מקור" — דגימות אמיתיות גם מטיקטוק/אינסטגרם (לא רק Pexels), דגימות AI גם אחרי גריסת-פלטפורמה
- [x] **1.3** כיול הסתברויות (CalibratedClassifierCV) + מדדים כנים (precision/recall/FPR/Brier out-of-fold) + סף FPR≤5% שמור ב-meta. תוצאה: AUC 0.962→0.975, FPR 2.7%, Brier 0.054 ✅
- [ ] **1.4** שדרוג מודל הפריימים: אימון על פריימים אמיתיים במקום דאטה סינתטי
- [x] **1.5** גרסאות נעולות ב-requirements; המודל אומן מחדש עם sklearn 1.5.0 (גרסת הפרודקשן) — האזהרה נעלמה ✅

**✔ מושלם כאשר:** המודל מאומן על ≥2,000 דגימות מכל הפלטפורמות, האימון רץ אוטומטית בענן כל לילה, והבנצ'מרק משלב 0 משתפר.
